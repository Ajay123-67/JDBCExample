JDBC CRUD Application (Java + MySQL)

This project demonstrates the implementation of JDBC (Java Database Connectivity) to perform basic CRUD operations (Create, Read, Update, Delete) using Java and MySQL.
Establishes connection between Java and MySQL database
Performs CRUD operations using PreparedStatement
Uses ResultSet to retrieve and display data
Menu-driven console application for user interaction
🔹 Technologies Used
Java
JDBC API
MySQL Database
🔹 Key Concepts Covered
Database connection using DriverManager
SQL query execution (executeQuery, executeUpdate)
PreparedStatement for secure and efficient queries
Handling ResultSet for data retrieval
🔹 Use Case
This project simulates a basic student management system where users can insert, view, update, and delete records from a database
